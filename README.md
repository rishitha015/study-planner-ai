\# Study Planner AI + Exam Scheduler



\## 📌 Project Description

Study Planner AI is a web application that automatically generates a personalized study timetable based on user subjects, available time, exam dates, and difficulty level.



\## ✨ Features

\- Add Subjects and Topics

\- Set Exam Dates

\- Auto Generate Study Timetable

\- Daily Study Plan View

\- Progress Tracking

\- Reminder Notifications (optional)



\## 🛠 Tech Stack

\- Frontend: React + Tailwind CSS

\- Backend: Node.js + Express

\- Database: MongoDB

\- AI Logic: Custom Scheduling Algorithm
